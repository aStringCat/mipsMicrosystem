# mipsMicrosystem
BUAA2024秋季学期计算机组成原理实验课程。ヾ(•ω•`)o


### P4

本次实验课程要求搭建一个单周期CPU。

能够支持简单的mips指令。

### P5

本次实验课程要求搭建一个五级流水线的CPU。

能够支持简单的mips指令。

### P6

本次实验课程在P5的基础上继续搭建一个五级流水线的CPU。

增加了更多的指令。

增加了乘除法模块，支持乘除法运算。

外置了IM和DM，相当于把内存从CPU中分离出来，作为外部接入设备。

### P7

增加了异常中断功能。

使用Bridge连接外部设备，突出低内聚高耦合的规范。

完成CP0的设计。
